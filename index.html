Weather prediction

This shiny application predicts weather in the Netherlands

The data used is from the Royal Dutch Meteorological Institution (KNMI). It contains data from 1900 up till 2015.

The predictors are:

  • Year
  • Month
  • Day

Slide 3

The outcomes are:

  • Duration of sunshine
  • Amount of precipitation
  • Duration of precipitation
  • Wind
  • Temperature

Note that the results are not the actual data on that day (if present in the source data), but a prediction based on a generalized linear model

Slide 4

Steps:

  • Read and prepare the data. The file knmi2.csv can be found on github in:

https://github.com/stevenzwart/Shiny-weahter-prediction

  • The data is loaded into a data frame
  • NA's are omitted

  • Month is treated as a factor: this is because high and low months are both cold. A linear model would not perform well if it was treated as numeric

  • Now create the linear models, one for each predicted variables. These are of the following form

Slide 5

library(caret)

glmtemp <- train(Temp ~ Year
              +Month
        +Day, data= dfsubknmi, method = "glm")

Now the application can be run. This shiny application can be found on github

https://github.com/stevenzwart/Shiny-weahter-prediction

Instruction are in the readme file.

The application has also been deployed on shinyapps.io

https://stevenzwart.shinyapps.io/Shiny6/

Note that the data is not useful if the dates are widely outside the range of measurements (between 1900 and 2015), so results are not meaningful if a date in 1700 or 2200 is inserted.
